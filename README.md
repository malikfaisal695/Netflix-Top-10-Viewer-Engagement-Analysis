# Netflix Top 10 Viewer Engagement Analysis

This project analyzes Netflix Top 10 weekly data to understand how rankings and IMDb ratings relate to viewer engagement and title longevity.

## Tools Used
- Excel – initial data inspection and cleaning
- MySQL – data cleaning and analysis
- Tableau Public – visualization and dashboarding

## Data Sources
- Netflix Top 10 weekly rankings
- IMDb ratings
- IMDb runtime data

## Analysis Questions
1. Do higher-ranked titles receive more weekly viewing hours?
2. Do IMDb ratings correlate with weekly viewing hours?
3. Do higher-rated titles remain longer in the Netflix Top 10?

## Key Insights
- Higher-ranked titles consistently receive more weekly viewing hours.
- IMDb ratings show a weak relationship with short-term weekly viewership.
- Higher-rated titles tend to stay longer in the Top 10, indicating stronger long-term engagement.
